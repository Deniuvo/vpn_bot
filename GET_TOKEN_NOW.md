# 🚀 Быстрое получение токена (ПОШАГОВО)

## ✅ Redirect URI настроен правильно: `https://yoomoney.ru`

---

## ШАГ 1: Запустите скрипт

```bash
python yoomoney_oauth.py
```

Скопируйте URL, который покажет скрипт.

---

## ШАГ 2: Откройте URL в браузере

Вставьте скопированный URL в адресную строку браузера и нажмите Enter.

---

## ШАГ 3: Авторизуйтесь и разрешите доступ

1. Войдите в свой аккаунт ЮMoney (если не авторизованы)
2. Прочитайте запрашиваемые разрешения
3. Нажмите кнопку **"Разрешить"** или **"Разрешить доступ"**

---

## ШАГ 4: СКОПИРУЙТЕ КОД (ВАЖНО!)

После авторизации вас перенаправит на страницу, и в адресной строке будет:

```
https://yoomoney.ru?code=ДЛИННЫЙ_КОД
```

**НЕМЕДЛЕННО скопируйте:**
- Весь URL: `https://yoomoney.ru?code=...`
- Или только код после `code=`

⚠️ **ВАЖНО:** Не уходите со страницы, пока не скопируете! Код быстро истекает!

---

## ШАГ 5: Запустите скрипт с кодом

Вернитесь в терминал и выполните:

```bash
python yoomoney_oauth.py "https://yoomoney.ru?code=ВАШ_КОД"
```

Или если скопировали только код:

```bash
python yoomoney_oauth.py ВАШ_КОД
```

---

## ШАГ 6: Сохраните токен

Если все успешно, скрипт покажет ваш токен доступа.

**Для Windows PowerShell:**
```powershell
$env:YMONEY_ACCESS_TOKEN="ваш_токен"
```

**Для Windows CMD:**
```cmd
setx YMONEY_ACCESS_TOKEN "ваш_токен"
```

**Для Linux/Mac:**
```bash
export YMONEY_ACCESS_TOKEN="ваш_токен"
```

---

## ✅ Готово!

После сохранения токена бот сможет автоматически проверять платежи через API ЮMoney.

---

## ⚠️ Если что-то пошло не так:

- **Код истек:** Коды действуют 1-2 минуты. Получите новый код
- **Ошибка:** Убедитесь, что скопировали код правильно и запустили скрипт быстро

---

**Удачи! 🎉**
