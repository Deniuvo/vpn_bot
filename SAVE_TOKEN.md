## ✅ Проверка токена

После сохранения проверьте, что токен установлен:

```powershell
python -c "import os; print('Токен установлен!' if os.getenv('YMONEY_ACCESS_TOKEN') else 'Токен НЕ установлен')"
```

## 🚀 После сохранения

После сохранения токена:

1. **Перезапустите терминал/PowerShell** (чтобы подхватились новые переменные окружения)
2. **Запустите бота:**
   ```bash
   python bot.py
   ```
3. **Проверьте логи при запуске** - должно быть:
   ```
   ✅ Токен API ЮMoney установлен. Автоматическая проверка платежей включена.
   ```
