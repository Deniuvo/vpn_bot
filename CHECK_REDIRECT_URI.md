# 🔍 Проверка Redirect URI

## Проблема: URL не принимается

Если при переходе по URL авторизации вы получаете ошибку `invalid_request`, это значит, что **redirect_uri в скрипте не совпадает с настройками приложения**.

---

## ✅ Решение: Синхронизировать redirect_uri

У вас есть два варианта:

### Вариант 1: Изменить в кабинете ЮMoney (рекомендуется)

1. Откройте: https://yoomoney.ru/myservices/new
2. Войдите в свой аккаунт
3. Найдите приложение **"Труба VPN"**
4. Откройте настройки приложения
5. В поле **"Redirect URI"** укажите:
   ```
   urn:ietf:wg:oauth:2.0:oob
   ```
6. **Сохраните** изменения
7. Подождите 1-2 минуты (настройки применяются с задержкой)
8. Попробуйте снова

### Вариант 2: Изменить в скрипте

Если вы не можете изменить настройки приложения, измените redirect_uri в скрипте на тот, который указан в кабинете:

1. Откройте `yoomoney_oauth.py`
2. Найдите строку 25:
   ```python
   REDIRECT_URI = os.getenv("YOOMONEY_REDIRECT_URI", "urn:ietf:wg:oauth:2.0:oob")
   ```
3. Замените на тот redirect_uri, который указан в кабинете, например:
   ```python
   REDIRECT_URI = os.getenv("YOOMONEY_REDIRECT_URI", "https://yoomoney.ru")
   ```

---

## 📋 Доступные варианты redirect_uri:

### 1. `urn:ietf:wg:oauth:2.0:oob` (для desktop приложений)
- **Плюсы:** Показывает код прямо на странице, не теряется
- **Минусы:** Нужно настроить в кабинете

### 2. `https://yoomoney.ru` (стандартный)
- **Плюсы:** Часто уже настроен по умолчанию
- **Минусы:** Код в URL, можно потерять

### 3. `http://localhost` (для разработки)
- **Плюсы:** Для локальной разработки
- **Минусы:** Работает только локально

---

## ⚠️ ВАЖНО:

**Redirect URI должен ТОЧНО совпадать** (включая регистр, протокол, слеши) между:
- Настройками приложения в кабинете ЮMoney
- Значением в скрипте `yoomoney_oauth.py`

Даже малейшее различие приведет к ошибке `invalid_request`!

---

## 🔄 После изменения:

1. Сохраните изменения
2. Подождите 1-2 минуты
3. Запустите скрипт заново:
   ```bash
   python yoomoney_oauth.py
   ```
4. Откройте новый URL в браузере

---

**Если проблема сохраняется:** Убедитесь, что изменения сохранены и прошло достаточно времени для применения настроек.
