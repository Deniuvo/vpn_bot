# 🔍 Как найти код авторизации после перенаправления

## ✅ Вы авторизовались и были перенаправлены на главный экран

Это нормально! Код авторизации должен быть в адресной строке браузера.

---

## 📍 Где искать код:

### Вариант 1: Проверьте адресную строку браузера

После перенаправления адресная строка должна выглядеть примерно так:

```
https://yoomoney.ru?code=abc123xyz456def789
```

или

```
https://yoomoney.ru?code=ВАШ_ДЛИННЫЙ_КОД
```

**Что делать:**
1. Посмотрите в адресную строку браузера
2. Найдите параметр `code=` в URL
3. Скопируйте ВСЕ, что идет после `code=` (до конца или до следующего `&`)

### Вариант 2: История браузера

Если вы уже ушли со страницы:

1. Откройте историю браузера (Ctrl+H)
2. Найдите запись `yoomoney.ru?code=...`
3. Скопируйте код из URL

### Вариант 3: Если код потерян

Если вы не нашли код и прошло много времени, коды авторизации одноразовые и действительны короткое время.

**Решение:** Запустите процесс авторизации заново:

1. Запустите скрипт:
   ```bash
   python yoomoney_oauth.py
   ```

2. Откройте новый URL в браузере

3. **ВАЖНО:** После авторизации НЕМЕДЛЕННО скопируйте URL из адресной строки!

4. Выделите и скопируйте весь URL, например:
   ```
   https://yoomoney.ru?code=abc123xyz456
   ```

---

## 🚀 Что делать дальше:

После того, как вы нашли код:

### Шаг 1: Скопируйте код

Вы можете скопировать:
- Только код: `abc123xyz456`
- Или весь URL: `https://yoomoney.ru?code=abc123xyz456`

Скрипт автоматически извлечет код из URL, если вы скопируете весь адрес.

### Шаг 2: Запустите скрипт с кодом

В терминале выполните:

```bash
python yoomoney_oauth.py ВАШ_КОД
```

**Пример:**
```bash
python yoomoney_oauth.py abc123xyz456
```

Или если скопировали весь URL:
```bash
python yoomoney_oauth.py https://yoomoney.ru?code=abc123xyz456
```

### Шаг 3: Получите токен

Если все правильно, скрипт покажет ваш токен доступа:

```
============================================================
✅ Токен успешно получен!
============================================================

Токен доступа:
4100119393589473-abc123...

============================================================
```

---

## 💡 Полезные советы:

1. **Копируйте сразу после перенаправления** - не уходите со страницы, пока не скопируете код
2. **Можно скопировать весь URL** - скрипт сам извлечет код
3. **Коды одноразовые** - каждый код можно использовать только один раз
4. **Коды быстро истекают** - используйте код в течение нескольких минут

---

## ❓ Что делать, если ничего не помогло:

Если вы не можете найти код:

1. **Запустите процесс заново** - это займет всего пару минут
2. **Будьте готовы** - сразу после авторизации скопируйте URL
3. **Используйте `urn:ietf:wg:oauth:2.0:oob`** - этот вариант показывает код на странице, а не в URL

Чтобы использовать другой вариант redirect_uri, измените в `yoomoney_oauth.py`:

```python
REDIRECT_URI = os.getenv("YOOMONEY_REDIRECT_URI", "urn:ietf:wg:oauth:2.0:oob")
```

---

**Главное:** Код должен быть в адресной строке сразу после перенаправления! 🔍
