# 💰 Учет комиссии ЮMoney для всех тарифов

## 📊 Расчеты для каждого тарифа:

### 1 месяц (90₽):
- **Пользователь платит:** 90₽
- **Комиссия (3%):** 2.7₽
- **Приходит на кошелек:** 87.3₽
- **Диапазон поиска (плавающий):** 83.81₽ - 90.79₽ (96% - 104% от суммы с комиссией)

### 3 месяца (250₽):
- **Пользователь платит:** 250₽
- **Комиссия (3%):** 7.5₽
- **Приходит на кошелек:** 242.5₽
- **Диапазон поиска (плавающий):** 232.8₽ - 252.2₽ (96% - 104% от суммы с комиссией)

### 1 год (800₽):
- **Пользователь платит:** 800₽
- **Комиссия (3%):** 24₽
- **Приходит на кошелек:** 776₽
- **Диапазон поиска (плавающий):** 744.96₽ - 807.04₽ (96% - 104% от суммы с комиссией)

---

## ✅ Формула работает универсально:

```python
amount_with_commission = payment_amount * (1 - YOOMONEY_COMMISSION_RATE)
# где payment_amount = 90, 250 или 800 (в зависимости от тарифа)

# Плавающий диапазон поиска (учитывает вариации комиссии 2-4% и округление)
min_amount = amount_with_commission * 0.96  # Минимум
max_amount = amount_with_commission * 1.04  # Максимум
```

---

## 🔧 Плавающий метод проверки

Система использует **плавающий метод проверки платежей**, который:

1. ✅ Учитывает стандартную комиссию 3%
2. ✅ Покрывает вариации комиссии от 2% до 4%
3. ✅ Учитывает возможное округление сумм
4. ✅ Ищет платежи в диапазоне ±4% от суммы с комиссией

Это гарантирует, что платеж будет найден даже если:
- Комиссия немного отличается (2.5% или 3.5%)
- Произошло округление при переводе
- Есть небольшие расхождения в сумме

## 🔧 Настройка комиссии (если нужно изменить):

На сервере можно изменить процент комиссии:

```bash
export YOOMONEY_COMMISSION_RATE="0.03"  # 3%
# или
export YOOMONEY_COMMISSION_RATE="0.025"  # 2.5%
```
