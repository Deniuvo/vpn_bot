# ✅ Проверка процесса оплаты и получения конфига

## 📋 Проверка процесса оплаты:

### 1. **Пользователь выбирает тариф** ✅
- Тарифы отображаются корректно
- Пользователь нажимает на кнопку тарифа

### 2. **Генерация платежа** ✅
- Создается уникальный комментарий для платежа
- Формируется ссылка на оплату через YooMoney
- Пользователь переходит на оплату

### 3. **Проверка оплаты** ✅
- Автоматическая проверка через YooMoney API
- Учитывается комиссия YooMoney (3%)
- Плавающая проверка суммы (диапазон 96-104% от суммы с комиссией)
- Поиск по комментарию и сумме
- Резервный метод через скриншот (если API недоступен)

### 4. **Генерация конфига WireGuard** ✅
- Генерируются ключи для клиента (приватный и публичный)
- Получается свободный IP адрес из диапазона 10.0.0.2-254
- Создается WireGuard конфиг (.conf файл)
- Конфиг сохраняется в папку `configs/`

### 5. **Добавление peer на сервер** ✅
- Peer добавляется на WireGuard сервер
- Peer сохраняется в `/etc/wireguard/wg0.conf`
- Пользователь сохраняется в БД с ключами и IP адресом
- Устанавливается дата окончания подписки

### 6. **Отправка конфига пользователю** ✅
- Файл конфига (.conf)
- Текст конфига для копирования
- QR-код для сканирования
- Инструкции по использованию
