# ⏰ Как работает истечение подписки

## ✅ Гарантия: После истечения срока VPN перестает работать!

### Как это работает:

1. **При покупке:**
   - Бот сохраняет дату окончания подписки (`expiry_date`) в базе данных
   - Например, для тарифа "1 месяц": `expiry_date = текущая_дата + 30 дней`

2. **При запросе конфига:**
   - Бот проверяет: `datetime.now() < expiry_date`
   - Если подписка истекла - конфиг **НЕ выдается**

3. **HTTP сервер конфигов:**
   - Проверяет активность подписки перед отдачей конфига
   - Если подписка истекла - возвращает ошибку 403 (Forbidden)

4. **Результат:**
   - ✅ Конфиг нельзя скачать через бота (`/myconfig`)
   - ✅ Конфиг нельзя скачать по прямой ссылке
   - ✅ Сообщение: "Подписка истекла, используй /buy для продления"

---

## 🔒 Защита на уровне сервера

### Проверка в боте:

```python
# В bot.py функция myconfig_command:
is_active = db.check_user_active(user.id)
if not is_active:
    # Конфиг НЕ отдается
    return "Подписка истекла"
```

### Проверка в HTTP сервере:

```python
# В config_server.py функция get_config:
if DB_AVAILABLE:
    is_active = db.check_user_active(user_id)
    if not is_active:
        abort(403)  # Forbidden - конфиг недоступен
```

## ⚠️ ВАЖНО: VPN автоматически отключается!

### ✅ **VPN полностью перестает работать после истечения срока!**

Для WireGuard система работает через **удаление peer с сервера**:

1. ✅ **При запросе `/myconfig` с истекшей подпиской:**
   - Peer автоматически удаляется с WireGuard сервера
   - VPN перестает работать **немедленно**
   - Пользователь деактивируется в базе данных
   - Пользователь видит: "🌙 Подписка истекла. VPN был отключен."

2. ✅ **Автоматическая проверка (скрипт `remove_expired_peers.py`):**
   - Скрипт находит всех пользователей с истекшей подпиской
   - Удаляет их peers с WireGuard сервера
   - Деактивирует в базе данных
   - VPN перестает работать для всех истекших пользователей

3. ✅ **Что происходит при удалении peer:**
   - Peer удаляется из работающего WireGuard интерфейса
   - Peer удаляется из конфига `/etc/wireguard/wg0.conf`
   - Конфиг на устройстве перестает работать (peer удален с сервера)
   - Пользователь деактивируется (`is_active = 0`)

**Результат:**
- ❌ VPN **не работает** сразу после истечения подписки (peer удален)
- ❌ Новый конфиг получить **невозможно** (подписка истекла)
- ❌ Прямая ссылка не работает (403 Forbidden)
- ✅ **Полная защита от неоплаченного использования**

## 🔧 Автоматическое удаление истекших peers:

Для **полной** защиты используется автоматическое удаление peers каждый день:

```bash
# Автоматическая проверка (через cron, каждый день в 3:00)
0 3 * * * /root/vpn_bot/remove_expired_peers.sh
```

Это обеспечивает:
1. ✅ Автоматическое удаление истекших peers каждый день
2. ✅ VPN перестает работать немедленно после истечения подписки
3. ✅ Конфиг на устройстве становится бесполезным (peer удален)
4. ✅ Защита даже если пользователь не запрашивал `/myconfig`
