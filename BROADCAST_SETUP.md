# 📢 Настройка рассылки через админскую группу

## 🎯 Как получить ID группы

### Вариант 1: Через бота @userinfobot

1. Добавьте бота @userinfobot в вашу группу `@winxvpn_admin`
2. Отправьте в группу команду `/start`
3. Бот покажет ID группы (это отрицательное число, например: `-1001234567890`)

### Вариант 2: Через бота @RawDataBot

1. Добавьте бота @RawDataBot в группу
2. Отправьте любое сообщение в группу
3. Бот покажет JSON с `"chat":{"id": -1001234567890}`
4. Скопируйте число из `"id"` (это ID группы)

### Вариант 3: Автоматически (если бот уже в группе)

Запустите временный скрипт:

```python
import os
from telegram import Bot

BOT_TOKEN = "8348229932:AAE414-MEiiDy6EQM3G0LH6OyawX-q8MuYU"
bot = Bot(token=BOT_TOKEN)

# Получите обновления
updates = bot.get_updates()
for update in updates:
    if update.message and update.message.chat.type in ['group', 'supergroup']:
        print(f"ID группы: {update.message.chat.id}")
        print(f"Название: {update.message.chat.title}")
        print(f"Username: {update.message.chat.username}")
        break
```

## ⚙️ Настройка на сервере

### Вариант 1: Через переменную окружения (рекомендуется)

```bash
export ADMIN_CHAT_ID="-1001234567890"
```

Добавьте в `~/.bashrc` для постоянного сохранения:

```bash
echo 'export ADMIN_CHAT_ID="-1001234567890"' >> ~/.bashrc
source ~/.bashrc
```

### Вариант 2: Прямо в bot.py (если не хотите использовать переменную)

Откройте `bot.py` и измените строку 31:

```python
ADMIN_CHAT_ID = os.getenv("ADMIN_CHAT_ID", "-1001234567890")  # Замените на ваш ID
```

## 📢 Использование рассылки

1. **Откройте группу** `@winxvpn_admin` в Telegram
2. **Отправьте команду** `/broadcast`
3. **Отправьте сообщение**, которое хотите разослать:
   - Текст
   - Фото (с подписью)
   - Видео (с подписью)
   - Документ (с подписью)
   - Любой другой тип сообщения
4. **Получите статистику** рассылки:
   - Сколько отправлено
   - Сколько ошибок
   - Всего пользователей

---

## 🔒 Безопасность

- ✅ Команда работает **только из админской группы** (если установлен `ADMIN_CHAT_ID`)
- ✅ Команду могут использовать **только администраторы группы**
- ✅ Обычные пользователи бота не могут использовать рассылку

```
Админ в группе: /broadcast
Бот: "Отправьте сообщение для рассылки..."

Бот: "✅ Рассылка завершена! Отправлено: 150, Ошибок: 2"
```

---

## ⚠️ Важно

- **Убедитесь**, что бот добавлен в группу как администратор
- **Проверьте ID группы** (должен быть отрицательным числом)
- Рассылка отправляется **всем пользователям** из базы данных
- Если пользователь заблокировал бота, отправка завершится ошибкой (это нормально)
