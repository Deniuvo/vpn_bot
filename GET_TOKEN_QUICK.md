# 🚀 Быстрая инструкция: Получение API токена ЮMoney

## Минимальные шаги (3 минуты)

### 1️⃣ Запустите скрипт
```bash
python yoomoney_oauth.py
```

### 2️⃣ Откройте URL в браузере
Скопируйте URL, который покажет скрипт, и откройте в браузере.

### 3️⃣ Авторизуйтесь и разрешите доступ
Войдите в ЮMoney и нажмите "Разрешить".

### 4️⃣ Скопируйте код из URL
После редиректа URL будет: `https://yoomoney.ru?code=ВАШ_КОД`  
Скопируйте значение `code`.

### 5️⃣ Обменяйте код на токен
```bash
python yoomoney_oauth.py ВАШ_КОД
```

### 6️⃣ Сохраните токен

**Windows PowerShell:**
```powershell
$env:YMONEY_ACCESS_TOKEN="ваш_токен"
```

**Windows CMD:**
```cmd
setx YMONEY_ACCESS_TOKEN "ваш_токен"
```

**Linux/Mac:**
```bash
export YMONEY_ACCESS_TOKEN="ваш_токен"
```
