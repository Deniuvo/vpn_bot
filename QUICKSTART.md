# Быстрый старт VPN Telegram бота

## Шаг 1: Установка зависимостей

```bash
pip install -r requirements.txt
```

## Шаг 2: Настройка WireGuard сервера

Убедитесь, что WireGuard установлен и работает на вашем сервере:

```bash
# Проверка статуса
sudo wg show

# Если не установлен (Ubuntu/Debian):
sudo apt update
sudo apt install wireguard wireguard-tools
```

## Шаг 3: Определение IP и ключей сервера

Запустите скрипт для автоматического определения настроек:

```bash
python setup_server.py
```

Скрипт покажет:
- IP адрес сервера
- Публичный ключ WireGuard (если доступен)

## Шаг 4: Настройка бота

### Вариант А: Через переменные окружения (рекомендуется)

```bash
export SERVER_IP="ваш_ip_адрес"
export SERVER_PUBLIC_KEY="ваш_публичный_ключ"  # опционально
```

### Вариант Б: Прямое редактирование

Откройте `bot.py` и найдите строки:

```python
SERVER_IP = os.getenv("SERVER_IP", "YOUR_SERVER_IP")
SERVER_PUBLIC_KEY = os.getenv("SERVER_PUBLIC_KEY", None)
```

Замените на:

```python
SERVER_IP = "ваш_ip_адрес"
SERVER_PUBLIC_KEY = "ваш_публичный_ключ"  # или None
```

## Шаг 5: Настройка HTTP сервера для конфигов

Бот автоматически запускает Flask сервер на порту 5000. Убедитесь, что порт открыт:

```bash
# Для Ubuntu/Debian
sudo ufw allow 5000/tcp
```

Для продакшена рекомендуется использовать nginx (см. README.md).

## Шаг 6: Запуск бота

```bash
python bot.py
```

Бот запустится и будет отвечать на команды в Telegram.

## Тестирование

1. Найдите вашего бота в Telegram по токену
2. Отправьте команду `/start`
3. Проверьте команды:
   - `/buy` - покупка подписки
   - `/status` - статус сервера
   - `/support` - поддержка

## Важные замечания

⚠️ **Безопасность:**

1. Никогда не публикуйте токен бота в публичных репозиториях
2. Используйте переменные окружения для чувствительных данных
3. Настройте файрвол для защиты портов
4. Используйте HTTPS для раздачи конфигов в продакшене

⚠️ **Проверка платежей:**

Текущая версия автоматически подтверждает оплату после получения скриншота.
Для продакшена рекомендуется:
- Интегрировать API ЮMoney для автоматической проверки
- Или настроить админ-панель для ручной проверки

- Проверьте открытые порты в файрволе
