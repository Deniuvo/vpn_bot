## ⚠️ Что нужно настроить перед запуском:

### 1. КРИТИЧНО: IP адрес сервера

```bash
# Через переменную окружения
export SERVER_IP="ваш_реальный_ip_адрес"

# Или отредактируйте bot.py, замените:
SERVER_IP = os.getenv("SERVER_IP", "YOUR_SERVER_IP")
# на:
SERVER_IP = "ваш_реальный_ip_адрес"
```

**Без этого бот не запустится!**

### 2. КРИТИЧНО: WireGuard сервер

Убедитесь, что WireGuard установлен и работает на сервере:

```bash
# Проверка
sudo wg show

# Если не установлен
sudo apt update
sudo apt install wireguard wireguard-tools
```

### 3. ОПЦИОНАЛЬНО: Токен API ЮMoney (для автоматической проверки)

Если хотите автоматическую проверку платежей:

```bash
# Установите библиотеку
pip install yoomoney

# Получите токен (см. GET_TOKEN.md)
python yoomoney_oauth.py

# Сохраните токен
export YMONEY_ACCESS_TOKEN="ваш_токен"
```

**Без токена бот будет работать через скриншоты!**

### 4. ОПЦИОНАЛЬНО: Публичный ключ WireGuard сервера

Если хотите автоматическое определение:

```bash
# На сервере
sudo wg show wg0 public-key

# Сохраните
export SERVER_PUBLIC_KEY="ваш_публичный_ключ"
```

**Если не указан, будет попытка определить автоматически.**

### 5. Файрвол и порты

```bash
# Разрешить порт для WireGuard
sudo ufw allow 51820/udp

# Разрешить порт для HTTP сервера конфигов
sudo ufw allow 5000/tcp
```
